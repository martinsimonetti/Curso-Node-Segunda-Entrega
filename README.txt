Las rutas para probar el sistema son:

http://localhost:8080/products/
http://localhost:8080/realtimeproducts/

La lista de productos, está vacía al principio. Desde la ruta de realtimeproducts se deben agregar productos para poder probarlo.
Las dos rutas están funcionando. Si al cargar un producto, falta un campo o se repite un código, se arroja un alert con el error.
